How It Works

A complete guide to the Archangel Command Center
What Is This Dashboard?

The Archangel Command Center is a central hub that tracks every marketing channel in one place — Google Ads, Google My Business, Search Console rankings, blog content, and competitor activity. It also surfaces prioritized action items so you always know what to focus on next.

North star goal: 30+ new client bookings/month by June 2026, with San Diego as the priority location.

Runs Automatically
No action needed for these
Blog Publishing Hourly

New blog posts go live automatically on their scheduled date. The scheduler checks every hour and publishes any post whose publish date has passed.

Run by: webflow-blog-scheduler/scheduler.py via cron

Dashboard Data Refresh Daily 6am

Search Console rankings and Webflow post counts sync automatically every morning. Data is cached locally so the dashboard loads instantly.

Run by: APScheduler background thread inside this app

Competitor Keyword Monitoring Weekly

A weekly report tracks competitor keyword usage across 15 tracked firms. New keywords and rising trends are surfaced in the Competitors tab.

Run by: automations/competitor-monitor/monitor.py

GMB Post Generator Weekly Sun 7am

Every Sunday, Claude generates 4 weeks of Google My Business posts (Mon=SD, Wed=Both, Fri=AV) and writes them to a Google Sheet queue. Posts are spaced 2–3 days apart and rotate through topic types: educational, local/geo, urgency, and service highlights.

Run by: automations/gmb-post-scheduler/post_generator.py via launchd

GMB Post Publisher Daily 9am

Every morning the scheduler checks the Google Sheet queue and publishes any post due that day to the correct GMB profile (SD, AV, or both). Status is written back to the sheet automatically.

Run by: automations/gmb-post-scheduler/scheduler.py via launchd

Google Ads Keyword Automation Daily

Scripts running inside Google Ads automatically pause keywords with CPA > $100 after 14 days and add negative keywords from poor search terms.

Paused until conversion tracking is fully verified.

Action Items Engine On refresh

Every time data refreshes, rules run automatically to generate the action items shown on the Overview tab — flagging high CPA, missing GMB posts, content gaps, and ranking opportunities.

Requires Manual Action
These need a human each time
Log Client Bookings As they happen

When a client books a consultation in Clio Grow, log it in the Bookings tab. Include location (SD or AV) and source (Google Ads, GMB, referral, etc.) to track what's driving conversions.

Future: Clio webhook will automate this

Upload Google Ads Report Weekly

Export the weekly performance report from Google Ads (or forward the automated email as HTML) and upload it in the Paid Ads tab. This populates spend, CPA, and keyword performance charts.

How to export: Google Ads → Reports → Predefined → Keyword → Download CSV

Generate New Blog Posts As needed

The content engine has 530 pending keywords ready. When the draft queue runs low (fewer than 8 scheduled), run a new batch.

cd archangel-marketing/seo-content-engine
python3 engine.py generate "keyword here"
python3 engine.py publish --all
python3 engine.py schedule
Review & Act on Action Items Weekly

Check the Overview tab weekly. Critical items (red) need same-day attention. Warnings (yellow) should be resolved within the week. Opportunities (blue) can be batched.

Tab Guide
Overview

Your daily check-in. Shows KPI scorecards vs. monthly targets, a goal progress bar, and the top 5 action items across all channels.

Check: Daily
Paid Ads

Google Ads spend, CPA, and keyword performance. Upload your weekly CSV or HTML report to keep this current. CPA target is <$150 (April), dropping to <$100 by June.

Check: After each upload
GMB

Live Google My Business metrics for both San Diego and Apple Valley: calls, website clicks, direction requests, and map views. Data pulled daily from the Business Profile Performance API.

Check: Weekly
SEO & Rankings

Live Google Search Console data — top organic queries, click/impression counts, and average position. Queries with >100 impressions ranking below #10 are highlighted as optimization opportunities.

Check: Weekly
Content Pipeline

Tracks all 542 target keywords (pending → generated → published), shows the draft queue scheduled in Webflow, and lists all live blog posts. When fewer than 8 drafts remain, generate a new batch.

Check: When queue gets low
Competitors

Keywords being used by 15 tracked competitor firms, sourced from the SEO analyzer database and weekly diff reports. Content gaps (keywords competitors rank for that Archangel doesn't) feed directly into action items.

Check: Weekly
